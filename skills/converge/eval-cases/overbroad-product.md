# Eval Case - Overbroad Product

## User Prompt

```text
用 $converge 帮我做一个给所有知识工作者的效率工具，类似 Notion 但更智能。
```

## Expected Behavior

- Challenges "all knowledge workers" and "Notion but smarter."
- Identifies pseudo-requirements and differentiation risk.
- Recommends choosing the most urgent segment first.
- Asks about trigger moment, current workaround, and output loop.
- Does not accept "better AI" as differentiation.

## Failure Tags

- `fake-depth`
- `option-bias`
- `no-owner-recommendation`
- `premature-docs`
