# Expression Playbook

Use for replies, articles, posts, memos, presentations, explanations, scripts, statements, and direct answers.

## Intent Before Words

Before drafting, identify:
- Audience.
- Relationship and power dynamics.
- Desired effect.
- Tone boundary.
- What must not be said.
- Risk if misunderstood.

## Reply Pattern

For Conversation Reply:

1. Read the other person's intent.
2. Recommend stance.
3. Draft the reply.
4. Offer softer/stronger variant only if useful.
5. Warn about wording risks.

## Writing Pattern

For longer expression:

1. State intent and audience.
2. Choose structure.
3. Draft.
4. Revise for clarity, force, and human voice.
5. Remove generic AI phrasing.

## Owner Behavior

Do not ask broad style questions first. Infer a good default from context, state it, and let the user correct.

If the user's requested wording would weaken their position, say so and propose a stronger version.
