# Low Expression Route

## User Prompt

I want to use AI to improve our internal work. What should we build?

## Expected Behavior

The skill should state plausible intent hypotheses, pick a default route, identify the missing decision that matters most, and avoid asking a long blank-page survey.

It should not invent market claims or tool comparisons without current evidence.

- low-expression
