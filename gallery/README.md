# Gallery

The Gallery will show before/after examples that make Converge's value legible in seconds.

Planned sections:

- vague startup idea
- messy personal or business decision
- current technical stack choice
- hard reply draft
- architecture handoff
- instruction poisoning review
- proof-overclaim correction
- host adapter behavior

Each example should include:

- user input
- weak response pattern
- Converge response pattern
- why the Converge response is better
- proof boundary

The gallery is intentionally separate from the core skill so examples can be expanded without bloating the skill context.
